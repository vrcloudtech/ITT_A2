Machine Learning

Machine learning also known as AI(Artificial Intelligence) is the process of making a program which adapts to the way it’s used, for example autocorrect is a form of machine learning were the program compares commonly used words to words being entered it also recommends words before being entered , all of this is done based on the users previous inputs and the most commonly used words for the corresponding user, this information is collected and stored for later references then once enough data is stored the program compares the data to find the most commonly used phrases and word storage by the user.

Machine learning is a constantly developing technology, the state of the art for this technology is a wide subject as machine learning can be categorised into both supervised and unsupervised machine learning algorithms and within these two categories machine learning can function in different ways such as the effects of machine learning within voice assistant technology such as apple's siri or amazon's alexa where as another instance in which machine learning may differ is through the usage of graphical recognition software such as facial recognition or iris recognition.

Supervised machine learning is when you have a user input stored and a prediction made for the output based on the previous results of the machine, the functionality of supervised machine learning is ended once the machine or program reaches a satisfactory ratio of correct output predictions and is able to re-use the collected data.

Unsupervised machine learning is when you have input data but do not create an output based on that data but instead use that data as a measurement for comparison as there is no correct answer to this data as this data is used to create recommendations for example the data a customer creates from looking at products can be compared to similar product data to generate what advertisements a customer may be interested in.

In this current time machine learning is used for small things such as auto correction functions, advertisement filtering and data collection but is also used for much larger programs such as automated response systems and graphic recognition software.

Within the next 3 years machine learning is likely to advanced greatly particularly in the medical industry with machines which are beginning to predict a patient's death date based off health history and more, according to Forbes the results of this testing reached a 95% accuracy, more can be found in this Forbes article here: https://www.forbes.com/sites/danielnewman/2018/07/23/three-ai-and-machine-learning-predictions-for-2019/#593f34124948.

Machine learning is becoming the future as it allows our applications that are already accurate to become more and more accurate and provide faster data analysis as it provides data which enhances our algorithms as we can isolate potential solutions with the extra data created via machine learning for example rather then having 10,000 potential solutions we can quickly re-evaluate our options to searching/generating a smaller array of potentially 1,000 or less solutions as we can predict what the user is attempting to do and find a solution faster.

A potential use for machine learning is interpreting slang phrases for voice activation as once the slang is recognized verbally it can interpret things as certain meanings and depending on the data collected it can eventually find the context the slang is used in as well as variations of the particular slang and in what situations it is more commonly used for example a check on potential slang usage in Australia against the uk and what is more commonly used in context there compared to Australia to find the most accurate description.
